## Extension Structure

A Zed extension is a Git repository that contains an `extension.toml`:

```toml
id = "my-cool-theme"
name = "My Cool Theme"
version = "0.0.1"
schema_version = 1
authors = ["Your Name <you@example.com>"]
description = "My cool theme extension"
repository = "https://github.com/your-name/my-zed-theme"
```
